Wordle

Project Overview

Wordle is a word puzzle game that challenges players to guess words based on the frequency of their letters.


I designed and developed a web-based version of Wordle as part of a class project. My goal was to demonstrate my proficiency in web development and put my theoretical understanding into practice. I utilized JavaScript and Bootstrap techniques to create a responsive design, providing me hands-on experience applying these technologies.

The project offered a valuable opportunity to reinforce my knowledge of JavaScript and responsive design through practical application.

Role UX, UI, Programming

Time3 weeks

Design Process

Develop

As a web developer, I began my journey by prototyping my project with HTML and CSS, which provided a strong foundation for my work. HTML allowed me to structure the content of my single-page application, while CSS enabled me to add visually appealing styles and elements.

Once I had established a solid foundation, I enhanced my project's functionality with JavaScript. This cutting-edge technology allowed me to add dynamic and interactive features to my application.
I implemented five main functions, including efficient storing and manipulating game data using arrays. I also incorporated modals to display game results and improve gameplay flow. Dynamic updates and user input validation were achieved through the creation and manipulation of the DOM.

To further improve the design and responsiveness of my application, I utilized the Bootstrap CSS framework, which enabled me to create a seamless user experience across multiple devices.

By combining HTML, CSS, JavaScript, and Bootstrap, I brought my vision to life and built an application that looked great and functioned smoothly and efficiently.
